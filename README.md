# Portfolio Anh Kiệt

Website portfolio cá nhân hiện đại với thiết kế glassmorphism, hiệu ứng 3D và animations mượt mà.

## Tính năng

- **Thiết kế hiện đại**: Glassmorphism, gradient background, dark mode
- **Animations**: GSAP animations, scroll reveal, hover effects
- **Responsive**: Tối ưu cho mobile, tablet, desktop
- **Interactive**: Custom cursor, parallax scrolling, skill bars animation
- **One-page layout**: Smooth scrolling giữa các sections

## Cấu trúc dự án

```
/
├── index.html          # File HTML chính
├── styles.css          # Styles tùy chỉnh
├── script.js           # JavaScript cho animations và interactions
├── assets/             # Thư mục chứa hình ảnh
│   ├── avatar.jpg      # Ảnh đại diện
│   ├── project1.jpg    # Ảnh dự án 1
│   ├── project2.jpg    # Ảnh dự án 2
│   └── project3.jpg    # Ảnh dự án 3
└── README.md           # Tài liệu này
```

## Cách chạy

1. Clone repository này
2. Mở `index.html` trong trình duyệt web
3. Hoặc deploy lên GitHub Pages

## Deploy lên GitHub Pages

1. Push code lên GitHub repository
2. Vào Settings > Pages
3. Chọn source là "main" branch và thư mục root
4. Truy cập URL được cung cấp

## Thêm hình ảnh

Thay thế các file trong thư mục `assets/` bằng hình ảnh thực của bạn:

- `avatar.jpg`: Ảnh chân dung của bạn (khuyến nghị: 512x512px)
- `project1.jpg`, `project2.jpg`, `project3.jpg`: Ảnh screenshot các dự án (khuyến nghị: 800x600px)

## Tùy chỉnh

- **Màu sắc**: Chỉnh sửa gradient trong `styles.css` và `index.html`
- **Nội dung**: Sửa text trong `index.html`
- **Animations**: Tùy chỉnh trong `script.js` với GSAP
- **Skills**: Thay đổi phần trăm trong HTML attributes `data-width`

## Công nghệ sử dụng

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Font Awesome (cho icons)
